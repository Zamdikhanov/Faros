Тип аксессуара


Степень защиты светильника


AKC002

(FF 101 L1,5 LIFT)

Тросовый подвес M4 (1,5м). Трос длиной 1,5 метра, имеет петлю на одном конце и микролифт на другом. Микролифт позволяет нажатием на кнопку регулировать длину троса. Также на конце микролифта имеется наружная резьба. Позволяющая крепить тросовый подвес к светильникам.

AKC004

(FF 90)

Скоба (переход на шпильку М8). Дает возможность светильник с креплением на тросовые подвесы, прикрепить с помощью скобы М8, либо просто крючком, либо петлей с тросом.

АКС010

(FF 50 BRACKET)

Кронштейн поворотный для FG 50, FG 55

AKC063

(FF 50 GRID 30-40 PACK)

Решетка защитная для FG 50 30W / 40W, FG 55 30W / 40W (L = 989 мм)

AKC064

(FF 50 GRID 45-55 PACK)

Решетка защитная для FG 50 45W / 55W, FG 55 45W / 55W (L = 1292 мм)

AKC065

(FF 50 GRID 75-100 PACK)

Решетка защитная для FG 50 75W / 100W, FG 55 75W / 100W (L = 1591 мм)
